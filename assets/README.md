# Assets

Use this directory for non-Markdown project assets.

## Folder Guide

- `board/` — Board art, map renders, layered map files
- `cards/heroes/` — Hero portraits and hero card art assets
- `cards/abilities/` — Ability card illustrations and icon composites
- `cards/items/` — Item card art and supporting visuals
- `tokens/` — Token sheets, token icons, and print graphics
- `icons/` — Shared UI icons and symbols
- `diagrams/` — Process diagrams, map callouts, and rules visuals
- `exports/` — Generated PNG, PDF, and print-ready output files
